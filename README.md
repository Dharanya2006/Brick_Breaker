## Brick Breaker Game (Java)
Description

This is a console-based Brick Breaker game developed in Java.
The game uses a 7×7 grid where a ball is launched to break bricks.

## Symbols Used

w → Wall

g → Ground

o → Ball

1,2,3… → Brick strength

## Commands

St → Move ball straight up

Lt → Move ball diagonally left

Rt → Move ball diagonally right

## Rules

The ball starts with 5 lives.

When the ball hits a brick, its strength decreases by 1.

If brick strength becomes 0, the brick disappears.

Ball life decreases only when a brick is hit.

If the ball hits a wall, it reflects.

If no brick is hit, the ball returns to the starting position.
